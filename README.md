| **Description**                                                                                                                                                                                                 |
|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| The Department of Consumer Affairs monitors the daily prices of 22 essential food commodities through 550 price reporting centres across the country. The Department also maintains buffer stock of pulses, viz., gram, tur, urad, moon and masur, and onion for strategic market interventions to stabilize the volatility in prices. Decisions for market interventions such as release of stocks from the buffer are taken on the basis of the price trends and outlook. At present, the analyses of prices are based on the seasonality, historical and emerging trends, market intelligence inputs, crop sowing and production estimates. ARIMA based economic models have also been used to examine and forecast prices of pulses. |

# Commodity Price Forecasting Application

This project is a full-stack web application that provides an interface for accessing and predicting commodity prices. It features a React frontend for user interaction and a Flask-based backend API for data processing and forecasting. The application uses time series analysis with the SARIMA model to forecast future prices of commodities such as Wheat, tur etc.

## Features

- **React Frontend**: A responsive and interactive user interface for visualizing commodity data and predictions.
- **Flask Backend API**: Provides endpoints for accessing commodity data, statistics, and predictions.
- **Commodity List**: Retrieve a list of available commodities.
- **Historical Data Access**: View historical price data for selected commodities.
- **Statistical Summaries**: Get insights into current, average, highest, and lowest prices.
- **Price Prediction**: Forecast future prices using advanced time series models.

## Technologies Used

- **React**: For building the frontend user interface with modern JavaScript.
- **Flask**: A lightweight web framework for the backend API.
- **Pandas**: For data manipulation and analysis in Python.
- **NumPy**: For numerical operations in Python.
- **Statsmodels**: For statistical modeling, specifically SARIMA for time series forecasting.
- **Flask-CORS**: To handle Cross-Origin Resource Sharing (CORS) for API requests.
